# VOIDWINK

## NO ALIGNMENT / NO NON-ALIGNMENT

A static site exploring the paradoxes of existence, alignment, and the void.

### About

This is a single-page static site featuring interactive elements and glitch aesthetics. The site explores themes of duality, non-duality, and the spaces between through five main sections:

- **Prologue**: Introduction to the void
- **Sutra**: Philosophical explorations
- **Manifesto**: Core principles
- **Glitch**: Technical philosophy
- **Epilogue**: Closing reflections

### Features

- Interactive subtitle system with easter eggs
- Glitch effects and text animations
- Collapsible sections for enhanced reading
- Responsive design
- Multiple easter eggs throughout (click around and explore!)

### Local Development

To run locally, simply open `index.html` in your browser, or use a simple HTTP server:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (if you have http-server installed)
npx http-server
```

Then navigate to `http://localhost:8000`

### Deployment

This site is designed to be deployed on GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in your repository settings.
